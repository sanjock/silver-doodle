공공 정보시스템 구축 제안

기술 스택: Nexacro N / eGovFrame / Tibero DB / MyBatis

01. 대국민 민원 통합 실시간 대시보드

시각화 대용량 처리 OSMU

개요: 전국 단위로 접수되는 민원 데이터를 실시간으로 수집하여 행정 지표를 시각화하는 시스템입니다.

  • 핵심 기능: 지역별 민원 밀집도(Heatmap), 처리 지연 알림, 부서별 처리율 비교 차트.
  • 개발 포인트: 넥사크로 Chart 컴포넌트를 활용해 동적인 화면을 구성하고, 티베로의 집계 쿼리를 이용해 대량 데이터를 빠르게 가공합니다.

02. 공공 자산 및 유지보수 이력 관리 시스템

데이터 정밀도 그리드 제어 Batch

개요: 공공기관 내 유·무형 자산의 취득부터 폐기까지 전 과정을 관리하고 유지보수 이력을 기록합니다.

  • 핵심 기능: 자산 바코드 연동, 유지보수 스케줄 관리, 수천 건의 자산 정보 일괄 수정(Multi-row Edit).
  • 개발 포인트: 넥사크로 Grid의 Dataset을 활용해 다중 행 편집을 구현하며, MyBatis의 foreach 태그로 티베로 DB에 일괄 저장합니다.

03. 지능형 전자결재 및 행정 워크플로우

보안 권한관리 트랜잭션

개요: 공공기관의 복잡한 결재 라인을 유연하게 설정하고 보안 가이드라인을 준수하는 전자결재 시스템입니다.

  • 핵심 기능: 동적 결재선 지정, 개인정보 마스킹 처리, 접속 이력 로그 감사.
  • 개발 포인트: eGovFrame Spring Security와 연동하여 메뉴 권한을 제어하고, 티베로의 트랜잭션 관리로 데이터 무결성을 보장합니다.
